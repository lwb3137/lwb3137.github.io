# 超簡單 GitHub Pages 個人頁（含 jQuery）

## 使用方式
1. 建立公開 repo：`你的帳號.github.io`
2. 上傳本專案的 `index.html` 到 repo 根目錄
3. 進入 repo：Settings → Pages → Source 選 Deploy from a branch；Branch 選 main；資料夾選 /
4. 等 1–3 分鐘，瀏覽 `https://你的帳號.github.io`

## 功能
- jQuery 顯示/隱藏「作品」
- jQuery 平滑回頂端
- jQuery + AJAX 讀 GitHub Repos（最多 5 筆）
